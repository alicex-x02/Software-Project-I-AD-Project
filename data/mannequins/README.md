# Mannequin Data

Sample mannequin PNG files are generated by:

```bash
python scripts/build_sample_db.py
```

The selector checks files in this order:

1. `{gender}_{age_group}.png`
2. `{gender}_unknown.png`
3. `unknown_unknown.png`
4. A generated dummy mannequin

Public age-group input is `adult` or `kids`; `kids` maps to the existing `*_child.png` mannequin files.
